# 交通管理在线服务系统 java824

## 项目概述

交通管理在线服务系统是一个基于 Spring Boot 和 Vue 的前后端不分离的系统。它提供了管理员和用户两端的功能模块，以便于进行有效的交通数据管理和业务处理。

## 技术栈

- 后端：Spring Boot
- 前端：Vue
- 运行环境：JDK 1.8 + MySQL 8
- 开发工具：Maven 3 + Navicat + IntelliJ IDEA

## 功能模块

### 管理员

- **管理员管理模块**
  - 添加、删除、修改管理员账号
  - 设置相应权限

- **基础数据管理模块**
  - 统一管理交通管理的基础数据
  - 包括道路信息、交通设施、车辆类型等

- **驾驶证业务管理模块**
  - 在线查询和增删改驾驶证业务信息

- **机动车业务管理模块**
  - 在线查询和增删改机动车业务信息

- **新闻信息管理模块**
  - 发布、编辑和删除交通新闻、政策更新、路况信息等

- **用户管理模块**
  - 管理用户账号，包括注册、登录、权限设置等

### 用户

- **个人中心管理模块**
  - 查看和编辑个人信息
  - 查看个人的交通记录

- **驾驶证业务管理模块**
  - 在线提交和查询驾驶证业务

- **机动车业务管理模块**
  - 在线提交和获取车辆业务办理结果

- **违法处理业务管理模块**
  - 在线查询和缴纳罚款、申请行政复议等

- **新闻信息查看模块**
  - 浏览交通新闻、政策更新、路况信息等

## 系统角色

系统分为管理员和普通用户两种角色。
